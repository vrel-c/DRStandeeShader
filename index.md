# 概要
『DRStandeeShader(Depth-Raised Standee Shader) / タテカンシェーダ』は、1枚の画像を貼ったQuad（板ポリゴン）を、シェーダー処理のみで「厚みのある立て看板（スタンディ）」のように見せることができる、Unity／VRChat向けシェーダーです。
