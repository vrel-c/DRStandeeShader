# 通常版について
**DRStandeeShader / タテカンシェーダ** <small>（通常版）</small>のフォルダ構造、および基本的な利用方法についてのマニュアルです。

## インポート後のフォルダ構造
通常版が正常にインポートされている場合、`Assets` フォルダ内に以下のフォルダ構造が生成されます。

```text
Assets/
└── VRElectroCenter/
    └── DRStandeeShader/              <- パッケージのルートフォルダ
        ├── Data/
        │   ├── Material/             <- サンプルマテリアル
        │   │   └── Texture/          <- サンプル用テクスチャ
        │   │       ├── Mask/         <- サンプル用マスク用画像
        │   │       ├── MatCap/       <- サンプル用マットキャップ
        │   │       └── NormalMap/    <- サンプル用ノーマルマップ
        │   ├── Script/
        │   │   ├── CSharp/           <- C#ソースコード
        │   │   └── Udon/             <- Udonプログラム
        │   └── Shader/               <- シェーダー本体
        │       └── Editor/           <- カスタムGUI
        └── Prefab/                   <- サンプルPrefab
```

* **`DRStandeeShader/`**
  * 本パッケージのルートフォルダです。このフォルダ直下に、表示サンプルのPrefabが含まれています。

* **`DRStandeeShader/Data/`**
  * シェーダー本体や、サンプル用のデータ（マテリアル、テクスチャ、スクリプトなど）が格納されています。
  * 通常、このフォルダ内のファイルを直接編集・変更・移動する必要はありません。

* **`DRStandeeShader/Prefab/`**
  * 本シェーダーと付属スクリプトを組み合わせた、サンプルPrefabが格納されています。
  * 各サンプルPrefabの利用方法や機能については、本ページ後半の「サンプルPrefabについて」を参照してください。
