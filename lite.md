# Lite版について

## インポート後のフォルダ構造
Lite版が正常にインポートされている場合、`Assets` フォルダ内に以下のフォルダ構造が生成されます。

```text
Assets/
└── VRElectroCenter/
    └── DRStandeeShader_Lite/  <- パッケージのルートフォルダ（サンプルPrefabを同梱）
        └── Data/              <- シェーダーファイル・サンプルマテリアルを格納
```
* **`DRStandeeShader_Lite/`**
  * 本パッケージのルートフォルダです。このフォルダ直下に、表示サンプルのPrefabが含まれています。
* **`Data/`**
  * シェーダー本体や、サンプルPrefab用のマテリアルが格納されています。<br/>
    <small>※ 通常、この中のファイルを直接編集・移動する必要はありません。</small>

## シェーダーの利用方法（初心者向け）
<small>※ Unityでのマテリアル作成やシェーダー割り当ての操作に**慣れている方は、このセクションを読み飛ばして**次の「マテリアルの設定項目（プロパティ）」へお進みください。</small>

本シェーダーは、基本的にUnityの **Quad** オブジェクトへ適用して使用することを想定しています。

### 1. マテリアルの作成と適用
1. **Project** ウィンドウの空いている場所を右クリックします。
2. メニューから **Create > Material** を選択し、任意の名前（例: `MyStandeeMaterial`）をつけます。
3. 作成したマテリアルを選択し、**Inspector** ウィンドウ最上部の **Shader** ドロップダウンをクリックします。
4. リストから **VRElectroCenter > DRStandeeShader_Lite** を選択します。

### 2. Quadオブジェクトへの適用
1. **Hierarchy** ウィンドウの空いている場所を右クリックし、**3D Object > Quad** を作成します。
2. 設定が完了したマテリアルを、作成した **Quad** へ直接ドラッグ＆ドロップすれば完了です。
   ※ イラストのアスペクト比（縦横比）に合わせて、Quadの `Scale` を調整してください。

## マテリアルの設定項目（プロパティ）

本シェーダーを適用したマテリアルの **Inspector** ウィンドウに表示される設定項目の一覧です。

<small>※ Lite版はカスタムGUI非搭載のため、Unityの標準プロパティ形式で表示されます。</small>

### Basic Settings（基本設定）
* **Depth Strength (0-1)** （厚みの調整）
  * 擬似的に描画する厚みの度合いを調整します。<br/>値を大きくするほど厚みが増しますが、処理仕様の関係等により側面部の描画が荒くなります。
* **Alpha Cutoff** （切抜きの閾値）
  * `Face Settings` 内 **Texture** の「透明な部分」を切り抜く基準値です。<br/>値を大きくするほど、より透明度（Alpha値）が低い境界部分まで削り取られます。
  *  切り抜かれたイラストの縁部分の形状が荒れていて気になる時などに調整してください。

### Face Settings（表情・顔設定）
* **Texture**（メインテクスチャ）
  * 正面側に表示される画像（透過PNG想定）を設定します。
  * 画像のタイリング（Tiling）およびオフセット（Offset）の調整が可能です。
  * **【重要】** Unityにインポートした画像の **Inspector** ウィンドウ（Texture Import Settings）にて、**Alpha is Transparency** のチェックが有効になっている画像を設定してください。
* **Back Color** （裏面の色）
  * 裏面側に表示される色を単色で指定します。<br />
    <small>※ 仕様上、カラーピッカーでアルファ値（透明度）を設定しても反映はされません。</small>
* **Side Color** （厚み部分の色）
  * 立体化された側面（厚み部分）の色を単色で指定します。
    <small>※ 仕様上、カラーピッカーでアルファ値（透明度）を設定しても反映はされません。</small>
    
::: warning 注意：Unity標準設定について
マテリアル下部に表示される **Render Queue などのUnity標準設定（Advanced Options）は、基本的には変更しないでください。**
本シェーダーはカットアウト描画での挙動を前提として設計されているため、値を変更すると正常に描画されなくなる原因となります。ただし、環境に合わせて `Enable GPU Instancing` を有効化するなどの調整は問題ありません。
:::
